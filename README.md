# NFT Marketplace

## Technology Stack & Tools

- Solidity
- Javascript
- Ethers
- Hardhat
- Ipfs
- React routers

## Setting Up

### Clone/Download the Repository

### Install Dependencies:

```
$ cd nft_marketplace
$ npm install
```

### Boot up local development blockchain

```
$ cd nft_marketplace
$ npx hardhat node
```

### Migrate Smart Contracts

`npx hardhat run src/backend/scripts/deploy.js --network localhost`

### Run Tests

`$ npx hardhat test`

### Launch Frontend

`$ npm run start`
